Task 1: jQuery effects: showing and hiding content.

What is jQuery?

jQuery is a JavaScript library. jQuery simplifies DOM modification and solves cross-browser problems.

How to get DOM elements with jQuery?

To get a specific element use $('selector') method (equivalent to jQuery('selector')). $() searches through the DOM for elements that match the provided selector and returns a new jQuery object that references these elements.

How to use $(document).ready() method?

.ready( handler ) method provides a way to execute JS code after the DOM is loaded. A handler is a function that will run only after the DOM is ready.

These equivalent syntaxes can be used:

  • $( handler )
  • $( document ).ready( handler )
  • $().ready( handler )

this with direct and delegated event handlers

For directly bound handlers this keyword is a reference to an element where handler was attached. Syntax for directly bound handlers: $().on('event', handler).

For delegated events this keyword is a reference to an element matching 'selector'. Syntax for delegated events: $().on('event', 'selector', handler).

Task 9: tab panels.

CSS width, height, max-width, margin and padding properties.

  • Width
  • Height
  • Margin
  • Padding

Width auto for block-level elements

The initial width of a block-level element is auto. This makes it expand to fit the horizontal space of the content area of the containing block.

Width 100% for block-level elements

If the element's width is set to 100%, that means that the element's total width will be calculated as 100% of the containing block width plus all horizontal paddings, borders and margins (unless the box-sizing: border-box is set, in this case only margins will be added to the 100% value).

By default elements' margins do not affect the containing block size. This can be changed by adding padding or border to the containing block.

When margin is set to auto the browser calculates margin value.

When margin is set in % it specifies the margin in % of the containing block's content area width.

When padding is set in % it specifies the padding in % of the containing block's content area width.
